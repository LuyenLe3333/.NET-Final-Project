# Restaurant Blazer App
Project is designed in **Blazer application** to track new restaurant reservations/receipts. Each page includes use of **C#**, **CSS**, **HTML** and **Bootstrap**.

## About Page
- Displays a short bio of Luyen Le
- Accessible from the top-right navigation bar

## Home Page
- Shows an introduction to the reservation/receipt tracker
- Styled with Bootstrap for a simple and professional format

## Reservation Page
- Allows input of **customer name**, **date**, **time**, **bill total**, and **payment status**
- Displays reservations in an **HTML table**
- Table can be filtered by **Reserved**, **Paid**, **Canceled**, or **View All**
